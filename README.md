# Galileo Finance

Plataforma fintech en Costa Rica para gestión de activos digitales y servicios financieros.

## Características

- Landing page responsive con Next.js 14
- Blog headless con WordPress
- Integración con i18n (Español/Inglés)
- Optimizado para SEO

## Desarrollo

```bash
npm install
npm run dev
```

Visita `http://localhost:3000` para ver el sitio localmente.

## Blog

El blog está integrado con WordPress Headless API. Ver `BLOG_INTEGRATION.md` para más detalles.
